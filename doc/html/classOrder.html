Detailed Description

Order aggregates all other classes except Shop.

This class is meant to represent an RobotModel Order. An Order is created by a SalesAssociate for a Customer and includes information about what the customer wishes to purchase from the Shop. Order also keeps track of the status of the Order via a Finite State Machine.
