Detailed Description

Arm is one of five RobotPart types.

A RobotModel can have multiple Arm components according the maxArms data member in Torso, another component of RobotModel. Arm only contains one unique data member, power, which represents the amount of power consumed when the RobotModel it is attached to is powered on.
